Having trouble verifying your email? We're here to help!

Frequently Asked Questions

I didn't receive the verification email

Check these locations:

  • Your spam/junk/promotions folder
  • All Mail or Trash folders
  • Social or Updates tabs (if using Gmail)

Still can't find it? Try resending the code from the verification page.

The verification code has expired

Verification codes expire after 15 minutes for security reasons. If your code has expired:

  • Click "Resend Code" on the verification page
  • Check your email for the new code
  • Enter the new 6-digit code within 15 minutes

The verification code is not working

Make sure you're entering the code correctly:

  • Enter all 6 digits without spaces or dashes
  • Check that you're using the most recent code
  • Make sure the code hasn't expired
  • Try copying and pasting the code from your email

I entered the wrong email address

If you registered with the wrong email address:

  • You'll need to register again with the correct email
  • Make sure to double-check your email before submitting
  • Contact support if you need help with account cleanup

Troubleshooting Steps

1

Check your email thoroughly

Look in spam, junk, promotions, and all other folders

2

Wait a few minutes

Email delivery can sometimes take 2-5 minutes

3

Resend the verification code

Click "Resend Code" on the verification page

4

Check your email settings

Make sure emails from pexilabs.com are not blocked

5

Contact support

If nothing works, reach out to our support team
