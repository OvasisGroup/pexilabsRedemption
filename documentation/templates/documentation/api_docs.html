Welcome to PexiLabs API Documentation

This documentation provides comprehensive information about integrating with the PexiLabs payment platform.

Quick Start

  • Authentication: All API requests require proper authentication using API keys
  • Base URL: https://api.pexilabs.com (Production) or http://localhost:8001 (Development)
  • Content Type: All requests should use application/json

Available Endpoints

  • POST /checkout/make-payment/ - Create a payment session
  • GET /integrations/merchant/dashboard/ - Merchant dashboard
  • POST /auth/login/ - User authentication

Need Help?

If you need assistance with integration, please contact our support team or check out our integration guides.
