Getting Started

Follow these step-by-step guides to integrate PexiLabs payment processing into your application.

1. Authentication Setup

First, obtain your API credentials from the merchant dashboard:

curl -X POST http://localhost:8001/checkout/make-payment/ \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 100.00,
    "currency": "USD",
    "customer_email": "customer@example.com",
    "description": "Test Payment",
    "callback_url": "https://yoursite.com/success",
    "cancel_url": "https://yoursite.com/cancel"
  }'

2. Processing Payments

Use the payment API to create payment sessions and process transactions.

3. Webhook Integration

Set up webhooks to receive real-time payment notifications.
